# Social Media Comment Analyzer - Client Setup Guide

## Step 1: Install Required Programs

Before starting, make sure you have these installed:

### 1. Python 3.8 or higher
- Download from: https://www.python.org/downloads/
- During installation, check "Add Python to PATH"

### 2. Node.js 14 or higher
- Download from: https://nodejs.org/
- Choose the "LTS" version

---

## Step 2: Install All Project Modules

1. **Double-click** the file:
   - `INSTALL_ALL_DEPENDENCIES.bat`

2. This will automatically:
   - Install all Python modules for the backend
   - Install all Node.js modules for the frontend
   - Show progress and any errors

3. Wait until you see the message:
   - "ALL DEPENDENCIES INSTALLED SUCCESSFULLY!"

---

## Step 3: Start the Application

1. **Double-click** the file:
   - `START_APP.bat`

2. Wait for the browser to open automatically
   - If it doesn't, open http://localhost:3000 manually in any browser

---

## Step 4: Use the Application

- Paste Instagram or Facebook post/group/profile URL
- Click "Analyze"
- View results, charts, and comment topics

---

## Troubleshooting

- If you see errors, read the messages in the black window
- Make sure Python and Node.js are installed
- Make sure your internet connection is working
- If browser doesn't open, open http://localhost:3000 manually
